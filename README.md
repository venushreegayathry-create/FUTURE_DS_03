# Marketing Funnel & Conversion Performance Analysis

## Objective
Analyze marketing funnel data to identify conversion drop-offs, understand funnel performance, and provide actionable recommendations to improve conversions.

## Tools Used
- Microsoft Excel (Pivot Tables, Charts)

## Approach
- Simulated a marketing funnel using available sales data
- Created funnel stages: Visitors, Leads, and Customers
- Calculated conversion rates between each stage
- Used pivot tables and charts to visualize funnel performance and drop-offs

## Key Insights
- Significant drop-off observed from visitors to leads
- Additional drop occurs from leads to customers
- Conversion rates remain consistent across the dataset

## Recommendations
- Improve strategies to convert visitors into leads
- Optimize lead-to-customer conversion process
- Enhance marketing campaigns to reduce funnel drop-offs

## Files Included
- Dataset (Excel file)
- Dashboard (Funnel chart)
- Report (Insights & Recommendations)
